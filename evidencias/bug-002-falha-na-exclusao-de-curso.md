# Bug 002 - Falha na exclusão de curso da listagem

## Descrição
A API não exclui o curso da lista quando clicado em excluir curso

## Passos para reproduzir 
1. Acessar Página de listagem de cursos
2. Selecionar um curso para exclusão
3. clicar em excluir curso
4. Deve aparecer uma mensagem de "Curso excluido com sucesso!"

## Resultado esperado
A API deve excluir o curso da listagem

## Resultado atual
O curso se mantem na lista

## Severidade
Média
